# MOYU 桌宠动画素材制作规范

这份文档可以直接发给像素动画师或桌宠美术，用于替换程序生成的内置皮肤。程序生成皮肤已经包含完整行为，但专业逐帧素材能明显改善角色的重量感、表演和辨识度。

## 基础规格

- 单帧画布：`48 x 48 px`，透明背景，角色脚底锚点保持在 `y=41` 附近。
- 输出方式：规则网格 spritesheet；每格严格 `48 x 48 px`，不得留额外间距。
- 程序当前读取 BMP spritesheet；同时交付带透明通道的 PNG 和 Aseprite 源文件，便于后续扩展加载器。
- 像素必须对齐整数坐标，不使用缩放插值、半透明描边、投影或背景。
- 所有动作保持相同视觉体积和底部锚点。位移动画由 Runtime 完成，素材只负责身体动作。
- 角色默认朝右绘制；Runtime 会自动水平镜像，尾巴、耳朵和表情需要允许翻转。

## 动作清单

每类至少 4 帧。优先把动作起因和性格表达画清楚，而不是单纯让身体上下移动。

| 动作 | 表演要求 |
| --- | --- |
| idle | 呼吸、重心轻微变化，循环接缝不可见 |
| blink | 快闭慢开，不能只是两张图切换 |
| sleep | 呼吸变慢、身体摊开，可带小型 Z 符号 |
| happy | 明显起跳和落地压缩 |
| sad | 耳朵或身体下垂，避免只加眼泪 |
| observe | 眼睛先转、头或身体后跟随 |
| walk | 有蓄力、迈步和落地重量 |
| work | 张嘴吞入、咀嚼或认真阅读 |
| wait | 克制地等待，偶尔看向旁边 |
| found | 先意识到，再惊喜，不是全程同一笑脸 |
| confused | 左右确认、停顿、问号反应 |
| giveup | 泄气并摊平 |
| bored | 发呆、走神、数点或轻微叹气 |
| sneak | 压低身体，目光锁定目标，小步接近 |
| scared | 先僵住，再缩紧或后撤 |
| playful | 假动作、坏笑、闪身 |
| peek | 身体探出，眼睛持续盯住目标 |
| dodge | 快速压缩、侧跳、刹停 |
| stretch | 拉长、停顿、放松回弹 |
| yawn | 闭眼、张嘴、身体放松 |

## 交付与验收

要求交付：

1. `skin.aseprite` 分层源文件。
2. 透明 `skin.png` 和程序用 `skin.bmp`。
3. 每类动作单独导出的 GIF 或 MP4 预览。
4. 一份帧序号到动作名的 Lua 映射。
5. 商用版权和二次修改授权说明。

找外包时可搜索“像素动画”“桌宠逐帧动画”“Aseprite 动画”。不要一开始购买整套素材，先付费试做 `idle`、`walk`、`scared` 三类：这三类能同时检验循环能力、重量感和表情演技。验收通过后再扩展剩余动作。避免购买只有换表情、没有身体重心变化的低价动作包。
